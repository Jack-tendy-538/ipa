# ipa
island ppt assistant

这是一个classisland插件，用于在classisland上控制正在放映的PPT文件。
## 功能
- 通过快捷键控制PPT的翻页（下一页/上一页）
- 通过快捷键开始或结束PPT放映
- 通过PowerPoint win32 API实现对PPT的控制
## 安装
1. 从[GitHub Releases](https://github.com/Jack-tendy-538/ipa/releases)下载最新版本的插件文件`ipa.cipx`。
2. 打开classisland设置，进入“插件”页面。
3. 点击“插件”，点击三个点选择`从本地安装`选择下载的`ipa.cipx`文件进行安装。
4. 安装完成后，重启classisland以启用插件。
